Simple plugin that will show related posts.

== Description ==
This is a Simple plugin that will show related posts under each single post depending on category. This doesn't have much option now but planning to build more option in future for this plugin

== Installation ==
Just Upload it using FTP or Using WordPress Dashboard.
Activate it 
And you are done

== Frequently Asked Questions ==
1. Q. Does this plugin has any option
Answer : No not yet. It will show 4 post under each single post based on category. No option for this plugin yet but have plan to add options like controlling the post numbers etc
